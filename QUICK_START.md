# Quick Start Guide - LeetCode Problem Tracker

## 🎯 Your Interview Prep Dashboard

Welcome! You now have a comprehensive tool for tracking your LeetCode problem-solving journey. Here's how to make the most of it.

## First Time Setup (2 minutes)

1. **Visit the Tracker**: Go to `http://localhost:3000/problems`
2. **Explore the Structure**: Scroll through all 9 stages
3. **Start Solving**: Pick a problem from Stage 1 and solve it
4. **Mark Complete**: Click the problem card to mark it done
5. **Watch Progress**: See your bars fill in real-time

## Daily Routine (15-20 min check-in)

### Every Day:

```
Morning: Solve 2 NEW problems
Afternoon: Re-solve 1 OLD problem
Evening: Review mistakes & update tracker
```

### Each Week:

```
Pick 1 day to review your progress
Check which categories you're struggling with
Adjust focus accordingly
```

## Managing Your Progress 💾

### Auto-Save

✅ Your progress is **automatically saved** to your browser

- No manual saving needed
- Works offline
- Survives browser restart

### Export (Backup)

```
Click "Export" button
→ Get problems-progress-YYYY-MM-DD.json
→ Keep it safe (cloud storage, email, etc.)
```

### Import (Restore)

```
Click "Import" button
→ Select your .json file
→ Progress restored instantly
```

### Reset (Fresh Start)

```
Click "Reset" button
⚠️ WARNING: This clears everything!
→ Confirm to proceed
→ All progress lost (use Export first!)
```

## Problem-Specific Tips 🎯

### Arrays & Hashing

- Master hash maps/sets first
- Two pointers for sorted arrays
- Sliding window for subarrays

### Sliding Window ⭐

- **VERY IMPORTANT** for interviews
- Identify expandable/contractable window
- Master this pattern!

### Trees 🌳

- Inorder/Preorder/Postorder traversal
- BFS with queue for level-order
- DFS with recursion
- **Spend serious time here!**

### Dynamic Programming 💣

- Break problem into subproblems
- Identify base cases
- Memoization (top-down)
- Tabulation (bottom-up)
- **Take this slowly!**

### Graphs 🌐

- BFS for shortest path
- DFS for connected components
- Union-Find for connectivity
- Must master BFS/DFS!

- Try importing from exported file

### Problems Not Loading?

- Refresh the page (Cmd+R / Ctrl+R)
- Check browser console for errors
- Restart development server
